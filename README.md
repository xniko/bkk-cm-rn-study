# bkk-cm-rn-study

React-native application to study react-native

## Package includes:

### React-native
https://facebook.github.io/react-native/

To get started with react-native. You need to install node, watchman and react-native-cli
https://facebook.github.io/react-native/docs/getting-started.html

### React
React-native apps are basicly react so you should get familiar with basic ideas of react.
https://facebook.github.io/react/tutorial/tutorial.html

### Redux
App is using Redux for state management
https://github.com/reactjs/redux

### React-redux
React-redux package connects react to redux
https://github.com/reactjs/react-redux

### react-native-router-flux
App uses react-native-router-flux for routings
https://github.com/aksonov/react-native-router-flux

### redux-logger
Redux middleware for console.logging redux store changes
https://github.com/evgenyrodionov/redux-logger
